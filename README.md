# lesson-OG03
Это учебный проект.

Небольшая игра ***Охота на уточек***, в 
которой есть следующие функции:
- стрельба по мишени с помощью мышки
- мишень появляется в произволной части экрана
- баллы за точные попадания
- раунды по 30 секунд
- отображение лучшего результата
